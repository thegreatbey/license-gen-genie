A simple CLI tool to generate open-source LICENSE files quickly.



---



\##  Install

```bash

npm install -g license-gen-genie

\# or use npx

npx license-gen-genie



---



\## Usage



Interactive Mode

```bash

npx license-gen-genie



Non-Interactive

```bash

npx license-gen-genie --license MIT



With Explanation

```bash

npx license-gen-genie --license GPL-3.0 --explain
